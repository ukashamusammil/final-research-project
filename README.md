## Project Overview
Real-time SIEM-based monitoring system for ESP32 IoMT devices.

## Components
1. Alert Prioritization (CRITICAL/HIGH/MEDIUM/LOW/INFO)
2. Alert Grouping (Attack campaign detection)

## ESP32 Devices Monitored
- Pulse Oximeter (MAX30102)
- Temperature Monitor (DS18B20/MLX90614)
- Fall Detection (MPU6050/MPU9250)
- ECG Monitor (AD8232)

## Dataset
- 100,000 samples
- 8 attack types
- 25 ESP32 devices
- 47 features
